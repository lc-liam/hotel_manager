# hotel_manager
Hotel management system making use of Object-Oriented Programming in Java.

Involves a few main classes: Admin, Client, Restaurant

Admin Functions
1. View Rooms
2. Create Rooms
3. View Dishes
4. Add Dishes
5. View Earnings

Client Functions
1. View Room Bookings
2. Book Rooms
3. Travel to restaurant
4. View Current Tab
5. Checkout

Restaurant Functions

1.displayMenu

2.addDish
